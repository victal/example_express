Example_express
---------------


Description
===========

This is a small application using express and it's
a work in progress.

How to Run
==========

* Install node.js (http://nodejs.org)
* Run 'npm install' to install dependencies
* npm start (to start with nodemon) or node app;
* Open http://localhost:3000

Tech Stack
==========

Currently using:

Node.js

Express.js 3.0

nodemon
